Milos’Shop – Serviços com Imagens Ilustrativas
Este documento organiza cada serviço do app Milos’Shop com a adição de imagens ilustrativas. O campo `imageUrl` deve ser adicionado em cada documento da coleção `services` no Firestore. Essas imagens podem ser trocadas por links do Unsplash, Pexels ou suas próprias fotos.
■ Estrutura no Firestore services: [
  {
    "id": "washVacuum",
    "name": "Ducha y aspirado",
    "description": "Incluye shampoo V-Floc",
    "price": {"auto": 50000, "suv": 70000, "camioneta": 100000},
    "imageUrl": "https://link-da-imagem-ducha.jpg"
} ]
■ Estilo CSS para as imagens .service-card img.service-img {
  width: 100%;
  height: 160px;
  object-fit: cover;
  border-radius: 8px;
  margin-bottom: 12px;
}
